# ⚡ CYBER NEXUS OS v3.0

מערכת הפעלה דמוית "האקר" (Mock OS) מבוססת Web, המשלבת סימולציות קוד קולנועיות, כלי ניטור רשת חזותיים וממשק משתמש רטרו-עתידני.



## 🚀 סקירה כללית
**Cyber Nexus OS** הוא פרויקט Front-end המדמה סביבת עבודה של אבטחת מידע. המערכת כוללת ניהול חלונות מורכב, אפקטים חזותיים של מסכי CRT, ומגוון "אפליקציות" פנימיות המדמות פעילות מערכת בזמן אמת.

## ✨ תכונות עיקריות
* **TyperScene Integration:** סימולטור הקלדת קוד התומך ב-Python, JavaScript ו-++C עם הדגשת תחביר (Syntax Highlighting).
* **מנהל חלונות (Window Manager):** תמיכה בפתיחה, סגירה, גרירה, מזעור והגדלה של מספר חלונות במקביל.
* **אפקטים חזותיים:** שכבת CRT עם "פליקר" (flicker), סמן עכבר מותאם אישית, ואנימציות Matrix.
* **סט של 9 אפליקציות מובנות:**
    * **Network Map:** מפה אינטראקטיבית של קשרי רשת.
    * **Threat Radar:** סורק איומים דינמי.
    * **System Monitor:** מעקב אחרי משאבי מערכת (CPU, RAM).
    * **Global Tracking:** גלובוס תלת-ממדי לניטור מיקומי תקיפה.
    * **Firewall:** תצוגה ויזואלית של חסימת חבילות מידע.
    * **Terminal:** שורת פקודה המדמה תהליך אתחול לינוקס.

## 🛠 טכנולוגיות
המערכת נבנתה בגישת **Vanilla Web** ללא ספריות חיצוניות כדי להבטיח ביצועים מקסימליים וטעינה מיידית:
* **HTML5:** מבנה סמנטי ושימוש ב-SVG עבור אייקונים.
* **CSS3:** שימוש ב-Variables, Flexbox, Grid, ו-Keyframe Animations מתקדמות.
* **Vanilla JavaScript:** תכנות מונחה עצמים (OOP) לניהול הלוגיקה של המערכת, ה-DOM וניהול החלונות.

## ⌨️ קיצורי מקשים
* `Alt + [1-9]`: פתיחה מהירה של אפליקציות מה-Taskbar.
* `Esc`: סגירת החלון הפעיל או עצירת ה-Typer.
* `Any Key`: הקלדה אוטומטית בתוך אפליקציית ה-Typer.
