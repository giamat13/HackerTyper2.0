# 🖥️ HackerTyper2.0

סימולטור מערכת הפעלה של האקר — מיועד לבדר, להרשים, ולבלבל כל מי שמסתכל על המסך.
כל הפעילות במערכת היא **מזויפת לחלוטין** ואין בה שום סיכון.

---

## 📁 קבצים

| קובץ | תיאור |
|------|-------|
| `index.html` | מבנה הדף וסרגל המשימות |
| `script.js` | כל הלוגיקה — חלונות, אפליקציות, קיצורים, מדריך |
| `styles.css` | כל העיצוב |

## 🎨 התאמה אישית

כל צבעי המערכת מוגדרים כ-CSS Variables בראש `styles.css`:

\`\`\`css
--primary: #0f0;        /* צבע ראשי — ירוק */
--accent:  #0ff;        /* הדגשה — ציאן */
--danger:  #f00;        /* אזהרה — אדום */
--warning: #ff0;        /* אזהרה משנית — צהוב */
\`\`\`

---

*פרויקט זה נוצר לצורכי בידור בלבד.*    * **System Monitor:** מעקב אחרי משאבי מערכת (CPU, RAM).
    * **Global Tracking:** גלובוס תלת-ממדי לניטור מיקומי תקיפה.
    * **Firewall:** תצוגה ויזואלית של חסימת חבילות מידע.
    * **Terminal:** שורת פקודה המדמה תהליך אתחול לינוקס.

## 🛠 טכנולוגיות
המערכת נבנתה בגישת **Vanilla Web** ללא ספריות חיצוניות כדי להבטיח ביצועים מקסימליים וטעינה מיידית:
* **HTML5:** מבנה סמנטי ושימוש ב-SVG עבור אייקונים.
* **CSS3:** שימוש ב-Variables, Flexbox, Grid, ו-Keyframe Animations מתקדמות.
* **Vanilla JavaScript:** תכנות מונחה עצמים (OOP) לניהול הלוגיקה של המערכת, ה-DOM וניהול החלונות.

## ⌨️ קיצורי מקשים
* `Alt + [1-9]`: פתיחה מהירה של אפליקציות מה-Taskbar.
* `Esc`: סגירת החלון הפעיל או עצירת ה-Typer.
* `Any Key`: הקלדה אוטומטית בתוך אפליקציית ה-Typer.
